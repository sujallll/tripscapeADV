# Tripscape Adventures

Premium cinematic travel website built with Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install
npm run dev
```

## Pages
- /
- /destinations
- /trips/[slug]
- /upcoming-trips
- /memories
- /about
- /testimonials
- /contact
